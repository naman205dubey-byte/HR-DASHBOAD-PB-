# HR Dashboard (Power BI)

An interactive **Power BI HR Dashboard** that provides insights into employee distribution, promotions, layoffs, job satisfaction, and workforce performance.

## Project Overview
This dashboard helps HR teams analyze employee data and monitor key workforce metrics to support data-driven decision making.

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Data Modeling

## Key Insights
- **Total Employees:** 1470  
- **Male:** 882 (60%)  
- **Female:** 588 (40%)  
- **Due for Promotion:** 72 (4.9%)  
- **Layoffs:** 117 (8%)

## Features
- Employee distribution by job level
- Promotion and layoff analysis by department
- Job satisfaction analysis
- Employee service year insights
- Work distance distribution
